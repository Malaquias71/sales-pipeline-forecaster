# Sales Pipeline Tracker

A lightweight, interactive web application to track, manage, and forecast sales deals. Perfect for small sales teams or individuals who want a clear view of their sales pipeline.

## Features

- **Add, Edit, and Delete Deals**  
  Quickly add new deals with deal name, value, and probability. Edit or remove deals easily.
  
- **Real-Time Metrics**  
  Automatically calculates total pipeline value, total expected revenue, and number of deals.
  
- **Expected Revenue Calculation**  
  Displays expected revenue per deal (`value × probability`) and sorts deals by expected revenue.
  
- **Persistent Storage**  
  Saves all deals in your browser using LocalStorage — data persists between sessions.
  
- **Sorting**  
  Sort deals by expected revenue to prioritize high-value opportunities.
  
- **User-Friendly Design**  
  Clean, modern UI with responsive layout and hover effects.

## Tech Stack

- HTML5  
- CSS3 (with modern styling and UX enhancements)  
- JavaScript (Vanilla JS, DOM manipulation, LocalStorage)

## Installation / Usage

1. Clone the repository:  
   ```bash
   git clone https://github.com/YOUR_USERNAME/sales-pipeline-tracker.git
   ```
2. Navigate to the project folder:  
   ```bash
   cd sales-pipeline-tracker
   ```
3. Open `index.html` in your browser.

No server required — fully client-side.

## How to Use

1. Fill in **Deal Name**, **Deal Value**, and **Probability (%)**.  
2. Click **Add Deal** to save it.  
3. To **edit a deal**, click the **Edit** button next to it, update the fields, and click **Update Deal**.  
4. To **delete a deal**, click the **Delete** button and confirm.  
5. Click **Sort by Expected Revenue** to order deals from highest to lowest expected revenue.
